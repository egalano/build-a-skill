## Tools and utilities

### `skill-creator` skill

- Built into Claude.ai, downloadable for Claude Code
- Can generate skills from natural-language descriptions
- Can review skills and suggest improvements
- Use: "Help me build a skill using skill-creator"
- Use: "Review this skill and suggest improvements"

Note: `skill-creator` helps you design and refine - it does not execute automated test suites or produce quantitative evaluation results.
